# **App Name**: GulfBridge HR

## Core Features:

- Homepage Showcase: Showcase key sections like 'Industries We Serve,' 'Categories of Workers,' and 'Countries We Supply' for immediate impact.
- Service Details Pages: Create individual pages detailing each service offered, such as Manpower Supply and Visa Processing.
- Manpower Catalog: Categorize and display available manpower types (Skilled, Semi-Skilled, Unskilled, Hospitality, Security) in a clear, catalog-like format.
- Country Information Hub: Provide detailed information for each Gulf country served, highlighting the legal process and benefits of hiring from India.
- Quote Request Form: Implement a contact form for companies to request manpower quotes, capturing Name, Company, Country, Requirement, and Contact details. Sends confirmation email.
- WhatsApp Integration: Integrate a WhatsApp button for direct inquiries.
- AI-Powered Content Improvement: Use an AI tool to optimize and rewrite the website content based on given user prompt.

## Style Guidelines:

- Primary color: Navy blue (#1A237E) to convey trust and professionalism.
- Background color: Light gray (#F5F5F5), almost white, to create a clean and corporate feel.
- Accent color: A slightly brighter blue (#5C6BC0), analogous to navy, for call-to-action buttons and highlights.
- Body and headline font: 'Inter', a sans-serif font for clear and modern readability.
- Use professional and clear icons related to industries served and worker categories.
- Maintain a clean and corporate layout with bold headlines and minimal text.
- Subtle transitions on image hover effects and button presses.
